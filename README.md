## Frontend Challenge
Frontend app that consumes the pokeapi.co API. Multi-language (english/spanish). Responsive design.

**Live: https://pokedex-asosa.vercel.app/**

### Used libraries:
- React.js
- Next.js
- next-translate
- Styled components
- TailwindCSS

### How to run locally:
1) Clone the repo
2) ```npm install```
3) ```npm run dev```

### Tests
```npm test```
